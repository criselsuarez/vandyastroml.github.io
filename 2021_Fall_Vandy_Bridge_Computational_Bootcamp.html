General Information

The main purpose of the workshop is to:

  • Introduce the Unix environment and bash commands
  • Teach the basics of Python, for beginners and more advanced users
  • Convey the idea of properly structuring your projects and code
  • Cover the basics of Git and Version control, as well as how to properly install them.
  • And more...

This workshop is split into two parts:

  • Website with slides and workshop information:
    https://tinyurl.com/bcb-2020
  • Repository with data from the workshop:
    https://tinyurl.com/bcb20-repo-main
  • To access the presentations for each topic, go the website shown above (https://tinyurl.com/bcb-2020)

Syllabus

  • Setting up Environments
  • Unix Tips and Tricks
  • Programming in Python
  • Structuring your Project
  • Version Control with Git
  • and more...
